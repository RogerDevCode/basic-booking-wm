## Técnicas adicionales que mejoran la profundidad

### Técnica 1 — "Sigue el hilo"
```
Para cada paper o doc que encuentres:
- Lee su sección "References" y sigue los 3 más relevantes
- Busca quién cita ese paper (Google Scholar → "Cited by")
- Rastrea la evolución del concepto en el tiempo
```

### Técnica 2 — "Perspectiva adversarial"
```
Además de buscar evidencia a FAVOR del tema,
busca activamente:
- Críticas documentadas
- Casos donde falló
- Alternativas que lo superan según sus propios autores
```

### Técnica 3 — "Audit de fechas"
```
Para cada fuente, reporta:
- Fecha de publicación
- Si hay versión más reciente disponible
- Si el contexto ha cambiado desde esa fecha
Descarta fuentes >3 años si el tema evoluciona rápido
```

### Técnica 4 — "Iteración por rondas"
```
Ronda 1: Busca los 5 recursos más autoritativos
Ronda 2: Por cada uno, sigue sus referencias clave
Ronda 3: Busca papers que citen los papers encontrados
Ronda 4: Busca issues/limitaciones conocidas
Ronda 5: Sintetiza y detecta contradicciones
```
